Sampo is now published to npm 🎉 For JavaScript/TypeScript projects, you can install it via `pnpm`/`npm`/`yarn`/`bun` without a Cargo toolchain, on Linux x64/arm64, macOS x64/arm64, or Windows x64.

```bash
pnpm add -D sampo   && pnpm sampo --help    # or:
npm  i   -D sampo   && npx  sampo --help    # or:
yarn add -D sampo   && yarn sampo --help    # or:
bun  add -D sampo   && bunx sampo --help
```
